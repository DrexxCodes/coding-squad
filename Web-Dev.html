Web Development Course Outline

The web development course operates the LAYP method. This means there's no specific duration the course will last. The duration actually varies with the individual taking the course. This means that you can decide when and where you'd have class within the subscription life-line and you can ask as many question as you like. The course has been very outlined so it's very easy to begin without a teacher. You'll just need nudges sometimes. The outline below is showing the things you'd learn with an average learning duration.
HTML
2 weeks

Learn HTML, the skeleton of the web. This tool will teach you fundamental concepts in Hyper Text Markup Language.

CSS
6 weeks

Cascading Style Sheet or CSS, is a styling ruleset that gives form and look to your website using colors, padding, margin, borders, content positioning etc.

JavaScript
13 weeks

JavaScript, arguably the most demanded language today is a very delicate language, though it's fun when you get a hang of it, it can be a little challenging. Arrays, loops, variables and constants may be a walk in the park. Asynchronous JavaScript, not so much.
